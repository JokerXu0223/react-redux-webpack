###版本环境
```html
node 6.0+(v6.13.1)
npm 3.0+(v3.10.10)
```

### 安装依赖
* Run 'yarn' OR 'npm install'

### 开发环境
* Run 'yarn start' OR 'npm run start'

### 部署环境
* Run 'yarn build' OR 'npm run build'
